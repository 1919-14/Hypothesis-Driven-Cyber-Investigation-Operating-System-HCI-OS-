# Report Assets

This directory contains assets used in report generation.

## Logo

Place the CERT-In logo here as `certin_logo.png`:

- **File**: `certin_logo.png`
- **Format**: PNG (transparent background recommended)
- **Size**: 800x240 pixels (or similar aspect ratio)
- **Usage**: Embedded in PDF cover pages and HTML reports

The logo you provided should be saved here as `certin_logo.png`.

## Other Assets

You can add other assets like:
- Custom fonts
- Chart templates
- Header/footer images
- Organization branding

These can be referenced in the report template configuration.
